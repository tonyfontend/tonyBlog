虽然图片是借的，是一整个圆，不过我只用了半个，没切···，还有浏览器只看了chrome···，太懒了···

首先他有一个一个永远在右侧的半圆，以及一个永远在动的半圆，当走刚开始的半圆的时候，隐藏右侧的半圆，当要走左侧的半圆的时候，显示右侧的半圆，恩··· 就是小于180度的时候，只显示那个在动的半圆，当移动超过180的时候，把右侧的半圆显示出来，在动的园继续大于180的运动，就ok了。

由于做了不同的时钟，所以把获取时间提成一个公共。

$.fn.getTime = function () {

    var date = new Date(),hour = date.getHours(),min = date.getMinutes(),sec = date.getSeconds();    
    return {hour : hour, min : min, sec : sec}
}

每个圆圈的html代码：

<div class="hour_area" data-flag="hour">
    <div class="left_clock_area">
        <div class="left_clock"></div>
    </div>
    <div class="left_bg"></div>
    <div class="right_clock_area">
        <div class="right_clock"></div> 
    </div>
    <div class="time_area"></div>
</div>

每个圆圈都是一样的，只不过各自加了一个data-flag,作为区别，js代码获取这个data-flag，然后获取getTime里面相应的数据做分析。

if(num > 30 ) {
    $leftBg.css('width' , '50%').hide();
    $rightClick.show();
}else if(num < 30 ){
    $leftBg.css('width' , '50%').show();
    $rightClick.hide();
}

然后在一个setInterval里去调取这些代码即可。
